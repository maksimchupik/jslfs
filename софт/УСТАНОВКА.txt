  🚀 БЫСТРАЯ УСТАНОВКА ЗАВИСИМОСТЕЙ
═══════════════════════════════════════════════════════════════

У вас есть 3 способа установить зависимости:

┌─────────────────────────────────────────────────────────────┐
│ СПОСОБ 1: Двойной клик на install.bat (САМЫЙ ПРОСТОЙ)      │
└─────────────────────────────────────────────────────────────┘

1. Найдите файл "install.bat" в этой папке
2. Дважды кликните на него
3. Дождитесь завершения установки
4. Готово!


┌─────────────────────────────────────────────────────────────┐
│ СПОСОБ 2: Запуск Python скрипта                             │
└─────────────────────────────────────────────────────────────┘

1. Откройте командную строку в этой папке:
   - Зажмите Shift и нажмите правой кнопкой мыши
   - Выберите "Открыть окно PowerShell здесь" или "Командная строка"

2. Введите команду:
   python install_dependencies.py

3. Дождитесь завершения


┌─────────────────────────────────────────────────────────────┐
│ СПОСОБ 3: Ручная установка через pip                        │
└─────────────────────────────────────────────────────────────┘

1. Откройте командную строку в этой папке

2. Введите команды по очереди:

   python -m pip install --upgrade pip
   python -m pip install aiogram>=3.0.0
   python -m pip install python-dotenv>=1.0.0
   python -m pip install telethon>=1.34.0
   python -m pip install openai>=1.0.0
   python -m pip install fastapi>=0.104.0
   python -m pip install uvicorn>=0.24.0
   python -m pip install pydantic>=2.0.0
   python -m pip install pyinstaller>=6.0.0

   ИЛИ одной командой:
   python -m pip install -r requirements.txt


═══════════════════════════════════════════════════════════════
  ⚠️  ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
═══════════════════════════════════════════════════════════════

1. Убедитесь, что Python установлен:
   - Откройте командную строку
   - Введите: python --version
   - Должна показаться версия (например, Python 3.11.0)

2. Если Python не найден:
   - Скачайте Python с https://www.python.org/downloads/
   - При установке ОБЯЗАТЕЛЬНО отметьте "Add Python to PATH"

3. Если pip не работает:
   - Попробуйте: python -m pip --version
   - Если не работает, переустановите Python

4. Если возникают ошибки при установке:
   - Попробуйте запустить командную строку от имени администратора
   - Или используйте: python -m pip install --user -r requirements.txt


═══════════════════════════════════════════════════════════════
  ✅ ПРОВЕРКА УСТАНОВКИ
═══════════════════════════════════════════════════════════════

После установки проверьте, что всё установлено:

python -c "import telethon; print('✅ Telethon OK')"
python -c "import fastapi; print('✅ FastAPI OK')"
python -c "import uvicorn; print('✅ Uvicorn OK')"
python -c "import openai; print('✅ OpenAI OK')"

Если все команды выводят "OK" - установка прошла успешно!
